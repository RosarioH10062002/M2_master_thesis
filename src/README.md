
## Scripts Overview

### `New_code.py`
**Trial + Baseline**

- Implements the trial version of the paradigm.
- Used to familiarize participants with the task.
- Includes a baseline block without stimulation.
- Ensures participants understand the task before the full experiment.

---

### `New_code_Only_Baseline.py`
**Baseline-only condition**

- Implements a baseline-only experimental condition.
- No isochronic tone stimulation is applied.
- Used as a control condition for behavioral and neural comparisons.

---

### `New_code_Complete.py`
**2AFC + Isochronic Tone Stimulation (ITS)**

- Full experimental paradigm.
- Combines:
  - a 2-Alternative Forced Choice (2AFC) task
  - Isochronic Tone Stimulation (ITS)
- Designed for data collection in the main experiment.
- Includes stimulus presentation, response logging, and timing control.

---
